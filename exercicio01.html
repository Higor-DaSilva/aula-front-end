Motivaçao

HTML (abreviação de Hypertext Markup Language) é uma linguagem de
marcação utilizada na esttrutura de páginas web. Sua sintaxe é bastante
simples e,assim como a XML é baseada em tags qu representam os
diversos elementos de uma página, como imagens e links.

Ao scessar uma página web atráves de um navegador, ele é capaz de
intrepetar o código HTML e renderiz[a-lo de foema compreeensível para o
usuário finsl, exibindo textos, botões, etc. com as configurações definidas por
meio das diversas tags que linguagem dispõe.
